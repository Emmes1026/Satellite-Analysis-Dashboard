# Satellite-Analysis-Dashboard

Educational project that includes Django and PyTorch for satellite photos analysis

Stack: Django, PyTorch, PostgreSQL, Docker

Satellite Analysis Dashboard is a full-stack web application designed for automated maritime surveillance. The system allows users to upload high-resolution satellite imagery and employs a custom-built Deep Learning model to detect ships and vessels in real-time.  The primary goal of this project was to explore the end-to-end pipeline of a Machine Learning product: from implementing neural network architectures in PyTorch to deploying a scalable web service using Django and Docker.
